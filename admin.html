🔥 Connect Firebase — 5 Easy Steps

Do this once. After setup, all your data will be saved permanently in Google's cloud.

1

Go to console.firebase.google.com and sign in with Google.

2

Click "Create a project" → Name it aarti-store → Continue → Disable Analytics → Create Project.

3

In the left menu click Firestore Database → Create Database → choose Test mode → select asia-south1 (Mumbai) → Enable.

4

Go back to Project Overview → click the </> Web icon → Register app as aarti-web → copy your firebaseConfig.

const firebaseConfig = { apiKey: "AIzaSyXXXXXXXXXXXXXX", projectId: "aarti-store", storageBucket: "aarti-store.appspot.com", ... };
5

Open admin/admin-firebase.js → Replace the firebaseConfig values with your own → Save → Re-upload.

✅ After setup your data will be stored in:
Firebase → Firestore → Collection: aarti_business → Document: aarti-db
